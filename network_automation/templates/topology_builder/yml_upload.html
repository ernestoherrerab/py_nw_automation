Topology Builder


Import your YML File


The file needs to be called hosts.yml !!!

The YAML file should follow the below format as is:

hostname:(without domain)
 groups:
  - ios_devices or nxos_devices
 hostname: 1.1.1.1 (IP if a DNS record does not exist)

YAML example:

cph-as1001:
 groups:
  - nxos_devices
 hostname: 1.1.1.1
cph-as1002:
 groups:
  - ios_devices
 hostname: 1.1.1.2
